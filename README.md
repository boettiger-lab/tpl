# Trust for Public Land Explorer

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18500623.svg)](https://doi.org/10.5281/zenodo.18500623)

Configuration for the **TPL — Protected Lands Explorer**, a conservation decision-support map app with a natural-language data agent. **Live:** <https://tpl.nrp-nautilus.io>

The map, chat, and agent code live in the [GLEN](https://github.com/boettiger-lab/geo-agent) library (loaded from CDN); this repo only carries the configuration — `index.html`, `layers-input.json`, `system-prompt.md`, and `k8s/`. See [AGENTS.md](AGENTS.md) to configure or deploy.
